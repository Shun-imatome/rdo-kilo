# rdo-kilo

###これはなに
RDO packstackでOpenStack Kiloの色々な環境を作る手順書のようなものです。といってもまだ書き始めたばかりです。

###環境について
以下の環境でPackstackコマンドを実行して環境を作ります。

- CentOS 7以降

そのうち、もう少しいろいろなディストリビューションで動作確認します。

OpenStack Compute上では上記以外のLinuxディストリビューションも動作します。
